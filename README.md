# Big old Button

A python iot project who makes you learn history events from Wikipedia via Twitter.

## Installation
  - Clone project
  - Intall dependencies
  - Rename `example.config.json` into `config.json` and fill with your infos and/or update values
  - `python index.py`

### Dependencies

  - Python 2.7.3 or older
  - 'requests' module
  - 'TwitterAPI' module
