# Big old Button

A python iot project who makes you learn history events from Wikipedia via Twitter.

## Installation
  - Clone project
  - `make install`
  - `make start`

### Requirements
  - Python 3.6.3 or higher
  - [pip](https://pypi.python.org/pypi/pip/) 9.0.1 or higher
